# Changelog

Alle nennenswerten Änderungen an dieser Foundation werden hier dokumentiert.

## [0.1.0] – 2026-05-22

### Erste Auslieferung
- Token-System mit über 100 CSS-Variablen (Farben, Spacings, Typografie, States, Form, Tag, Avatar)
- Schrift: Geist als einzige Schriftfamilie
- Basis-Stile in `ssf-foundation.css` (Typo-Stufen H1–H6, Body-small, Caption, Footnote, Blockquote, Inline-Link)
- Visuelle Dokumentation in `design-system-foundation.html` mit On-Page-Navigation
- States: Focus-Ring, Disabled, semantische Banner (Error / Success / Info)
- UI-Tokens: Form-Inputs mit Label/Helper/Error, Tags / Badges, Avatare (sm / md / lg)
- Logo-System mit definierter Schutzzone
- README mit Schnellstart, Naming-Conventions und Erweiterungs-Workflow
